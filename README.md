# SELF (Support‑First Logic Engine)

SELF is the internal library used by SoulsynqAI to enforce support-first, safety-oriented behavioral constraints.

- Source: `src/`
- Build output: `dist/` (committed; `npm run build` regenerates and patches ESM import specifiers for Node ESM)
- Docs: `SELF_DOCS/`

## Development

```bash
npm install
npm test
npm run build
```
